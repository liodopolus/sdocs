# Howto Setup Github

I Prerequisites

  1. git
  2. ssh
  3. login on github

II Installation for Slackware

  1. use slackpkg
  2. use sbopkg
  3. use SlackBuild scripts

III Configuration

  1. quick setup

create ssh key add ssh key create a repo in github push your changes to your repo

2. setup github https://help.github.com/articles/set-up-git/

see too man git or man gittutorial:

git config --global user.name "YOUR NAME"
git config --global user.email "YOUR EMAIL ADDRESS"
  1. initialisize repo and push it to remote repo with same name

cd repo touch README.md git init git add README.md git commit -m “first commit” git remote add origin git@github.com:githubuser/repo.git git push -u origin master
