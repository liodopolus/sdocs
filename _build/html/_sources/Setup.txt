I Prerequisites

1. server: nginx
2. framework: django
3. databank: postgresql (mariadb)
4. python-translator: gunicorn (uwsgi)

Addons
a. virtual environments: virtualenv 
b. documentation: sphinx
c. maybe pip for easy installation

II Installation for Slackware

1. use slackpkg
2. use sbopkg
3. use SlackBuild scripts
